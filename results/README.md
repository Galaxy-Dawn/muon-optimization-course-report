# Results

This directory contains raw CSV outputs generated by the scripts in `experiments/`.

- `matrix_quadratic_diagnostics.csv`: iterations to tolerance for the matrix quadratic experiment; values above the budget indicate nonconvergence or numerical divergence.
- `weight_decay_control.csv`: spectral-norm statistics for the decoupled decay recurrence under random and adversarial polar updates.
- `rms_alignment_shapes.csv`: observed exact-SVD matrix-sign RMS, theoretical RMS, maximum absolute discrepancy, and shape-scaled RMS across matrix shapes.
- `ns_approx_quality.csv`: Newton-Schulz alignment, orthogonality, and singular-value-spread diagnostics.
- `digits_logreg_results.csv`: five-split logistic-regression accuracies.
- `digits_mlp_results.csv`: five-split MLP optimizer ablation accuracies.
- `digits_mlp_stability.csv`: final MLP maximum logits and weight spectral norms.
- `classification_lr_sweep_raw.csv`: per-split validation-selected classification results.
- `classification_lr_sweep_summary.csv`: aggregate validation-selected classification diagnostics used in the course report.

- `matrix_quadratic_final_diagnostics.csv`: long-form final relative objective gaps and relative parameter errors for the matrix-quadratic runs.

- `finite_ns_training_raw.csv` and `finite_ns_training_summary.csv`: finite Newton--Schulz MuonW MLP training diagnostics for K in {1,3,5,8}, including accuracy, loss, orthogonality error, runtime, and spectral norms.
